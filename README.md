# Bone Fracture Detection using Deep Learning

## Overview
This project builds a deep learning model to detect bone fractures from x-ray images using a Convolutional Neural Network (CNN) and compares it with a pretrained ResNet model.

The model classifies images into two categories:
- Fractured
- Non-fractured

The goal is to demonstrate how machine learning can assist in medical imaging tasks and support data-driven diagnostics.

Disclaimer: This project should not be used as a replacement for professional medical diagnosis.

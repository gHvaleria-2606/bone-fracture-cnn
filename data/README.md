## Dataset
- This project uses the "X-ray Images of Fractured and Healthy Bones" dataset by Nayem et al. (2025), available on Mendeley Data.
- DOI: https://doi.org/10.17632/cd6jjsxz44.1 .
- Licensed under CC BY 4.0.

- The dataset was reorganized and preprocessed for this project. Only original images were used to avoid data leakage.

- You can download the dataset here: https://www.kaggle.com/datasets/foyez767/x-ray-images-of-fractured-and-healthy-bones/data

- After downloading, place the data in the following structure:
```
data/
|---original/
|   |---fractured/
|   |---non_fractured/
```
